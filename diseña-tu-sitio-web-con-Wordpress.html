    Diseña Tu sitio web con Wordpress

    Anteriormente te mostramos como puedes adquirir un hosting y un dominio para tu sitio web, en el caso que no lo hayas hecho dale clic aquí, ahora viene la parte más divertida, es el diseño, Wordpress es una herramienta que nos puede ayudar a realizar un diseño muy fácil y le lo vamos a mostrar.


    Si ya ingresaste al administrador de Wordpress, debes dirigirte a “Apariencia”, luego a “Temas”, y se te van a desplegar todos los temas que posee Wordpress. Existe una infinidad de temas gratuitos es cuestión que tu escojas uno que te guste, para este caso voy a escoger el “Mins”. Le das a "Añadir nuevo".



    En el buscador si ya conocen un tema puedes buscarlo.



    Si ya escogiste el tema le das a “Personalizar” y se te abre el tema y luego le das a “Instalar”.



    Listo ya tienes tu tema instalado, te vas a la parte izquierda donde puedes ver tus páginas, le das a añadir página.



    1.- Puedes añadir un título.

    2.- Puedes añadir texto y una descripción.

    3.- Puedes añadir imágenes.



    Mientras vas diseñando tu sitio web tienes una vista previa, y actualizas para ver los cambios.



    A continuación te dejo el nombre de algunos temas que podrían sentarle muy bien a tu negocio y que los puedes usar libremente.







    Listo hay algo que quisiera comentarte, todas estas plantillas de Wordpress están optimizadas para móviles, es decir tu página web se verá excelente en cualquier dispositivo móvil.



    Si quieres aprender a diseñar de mejor manera tu sitio web para hacerlo más llamativo y atraer tráfico te recomendamos nuestros cursos.
